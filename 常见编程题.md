一、实现curry函数
curry实际是一种函数的转换，能够改变函数调用的方式。如f(1,2,3)可以修改为f(1)(2)(3)的形式调用，curry本身不对函数进行调用。

```js
function curry(fn) {
  return function inner(...args) {
    if(args.length >= fn.length) {
      return fn.apply(this, args)
    }else {
      return (...arg2) => inner(this, args.concat(arg2))
    }
  }
}
```
